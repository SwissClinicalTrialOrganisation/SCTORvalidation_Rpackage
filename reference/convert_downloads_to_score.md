# Convert number of downloads to a score The score is based on the approach by the riskmetric package. It is based on a simplification of the logistic curve.

See [riskmetric
documentation](https://pharmar.github.io/riskmetric/reference/metric_score.pkg_metric_downloads_1yr.html)
for further details.

## Usage

``` r
convert_downloads_to_score(downloads)
```

## Arguments

- downloads:

  number of downloads
